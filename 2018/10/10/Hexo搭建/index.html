利用Hexo快速搭建博客(无图版)

发表于 2018-10-10

第一步：

  • Hexo第一步，github建仓库：我们首先得在github上创建一个仓库，仓库名字为：username.github.io（最好就用自己的username，要不然在上传hexo的时候配置很麻烦）

第二步：

  • 这一步，先不用管github。我们在本地新建一个空文件夹（随便在哪里），右键打开git bash（VScode的终端也可以，推荐）：
    • 首先要全焗安装一下hexo：

      npm install hexo -g

    • And then初始一下hexo，稍安勿躁，等一哈哈

      hexo init

    • 出现Start blogging with Hexo就表示你可以进入下一步了->安装所有的依赖：（如果之后你安装主体或者插件出现报错缺少依赖时也可以删除node_modules文件夹再重新执行一遍npm install）

      npm install

    • 这样你的Hexo就安好了，不过我们还需要创建一下渲染的文件，输入：

      hexo generate //或者hexo g就行了

    • 这样你的blog的html等文件就已经创建好了，现在我们可以在本地测试一下

      hexo server //或者hexo s 他会默认打开4000端口

    • 在本地输入localhost:4000

第三步：

  • 这也是关键步骤了，就是将我们的github的远程仓库与本地的hexo关联起来,进入hexo/_config.yml,在最后的deploy编写：

    npm install hexo-delpoyer-git –save
