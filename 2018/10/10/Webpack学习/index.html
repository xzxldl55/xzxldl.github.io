  • webpack.config.js为配置文件
  • 热更新webpack-dev-server
    • 下载

    • npm install webpack-dev-server –save-dev

    • 配置devServer
    • 要先在package.json配置一下script

    • “script”:{“server”:”webpack-dev-server”}

    • 再在终端输入npm run server可打开服务器
    • 可能会报错要安装一下webpack-cli

    • npm install –save-dev webpack-cli
